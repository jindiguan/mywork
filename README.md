## 初衷

2016年7月1日，本科毕业，正式离开校园生活，走向了名叫“社会”的历练场。

2019年1月7日，进入工作后的第三个年头，时光不断流逝，日复一日。回望这满打满算的950天，如果不仔细去记录，不断涌现在脑海中的，彷佛是模糊的同一天。 

为了梳理过去工作上的成长，也为了将自己的经历和学习到知识的分享给各位。
